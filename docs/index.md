# Claude Developer Wiki

Welcome to the internal wiki for using Claude at work. This wiki covers how to get set up, how each tool works, and how to get the most out of Claude as a developer.

## What's available

| Tool | What it is | Best for |
|---|---|---|
| [Claude.ai Web](getting-started/claude-web.md) | Browser-based chat interface | Quick questions, exploring ideas, sharing with teammates |
| [Claude Code (CLI)](getting-started/claude-code.md) | Terminal tool that works inside your project | Coding, debugging, refactoring, running commands |
| [Claude Cowork](getting-started/claude-cowork.md) | Desktop app feature for autonomous multi-step tasks | File generation, research, scheduled work |
